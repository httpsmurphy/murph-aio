Setup Guide

Option A — One Click (Recommended)

  1. Click "Open Chrome" above to launch Chrome with the extension loaded.
  2. Pin the Murph AIO extension in Chrome's toolbar.
  3. You're ready to go — create tasks in this app and they'll run automatically.

Option B — Manual Load

  1. Copy the extension path above.
  2. Open Chrome and go to chrome://extensions.
  3. Enable Developer mode (top right toggle).
  4. Click "Load unpacked" and paste the path.
  5. Make sure Murph AIO is running when you use Chrome.
